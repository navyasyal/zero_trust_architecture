# Zero Trust Node.js Application

## Project Description
This is a Node.js CRUD application integrated with Jenkins CI pipeline.

## Technologies Used
- Node.js
- Express.js
- Git
- Jenkins
- ngrok (for webhook testing)

## CI/CD Integration
This project is connected to Jenkins using:
- Pipeline script from SCM
- GitHub repository
- Webhook via ngrok

Every push to the master branch triggers a Jenkins build automatically.
